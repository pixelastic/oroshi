# x oroshi

This is my dotfiles repository. It contains most of my configuration as well as
convenient scripts to deploy, generate and more generally update my config.

Feel free to browse and borrow ideas from it.

## Note on my setup

I'm working on Ubuntu, with Gnome and Wayland.

I plan on migrating off Gnome in the future, but this is a big project so I
haven't committed yet. Sway or Hyperland seem nice alternatives.

The issues with Gnome are:
- Adding/updating extensions is a laborious process
- Stuff breaks from one version to the next
- The notify-send default system is buggy and chokes on queue messages and can't
  be updated.
