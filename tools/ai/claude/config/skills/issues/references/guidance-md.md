# guidance-md.md

A file that bridges context windows across sessions.
Split into two sections with different ownership rules.

```markdown
## Guidance

This is written once, and never modified by agents.

- Static instructions specific to this PRD.
- Testing commands
- File locations (relative to the repo root)
- Conventions
- Prior art references

## Discoveries

This is append-only, updated by agents after each issue.

### Issue XX — short title
- Non-trivial finding 1
- Non-trivial finding 2
```
